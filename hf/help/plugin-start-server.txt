
Display help for "start-server"

Description:
  Starts the application server with the domain.

Requirement in the configure file:
  server.home          The path to the application server
  domain.name          The name of the domain
  domain.home          The path to the domain directory.

Usage:
  $ hf start-server

Additional Arguments:
  -

Example:
  $ hf start-server
  $ hf start-server -v
  $ hf start-server -q

Note:
  -v | --verbose       display more information.
  -q | --quiet         display only errors or warnings
