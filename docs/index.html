Heringsfish Command Line Interface (cli)

Command Line Interface to manages the Glassfish / Payara Application Server and the deployment on local or server machine.

Overview

The Command Line Interface allows you to a GlassFish or Payara Application Server easy to use and a encapsulated development environment for your JEE 7 project.

Requirement

The following programs or modules are necessary:

  • Node JS: This cli is tested with the version 6.x or higher
  • Set the environment variable JAVA_HOME must be set (Java 8).
  • Application Server Glassfish 4.1 or Payara 4.1: Just install the server somewhere on your computer or in the project.
  • Maven: The command mvn should in your PATH.
  • An Java IDE :-)

Installation

Install global:

$ npm install -g heringsfish-cli
